## Technologies
Project is created with:
* ESP32 x 3
* Raspberry Pi x 1
* Module counting sensor x 3
* MG995 Servo Motor (180 Degree)  x 2
* MG995 Servo Motor (360 Degree) x 4
* PCA9685 16-Channel 12-bit PWM Servo shield I2C interface x 1
* Button switch module x 6
